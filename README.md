# MS SQL Datawarehouse Project
Building a modern  data warehouse MS SQL Server project, including ETL, data modeling and analytics

This project demonstrates a comprehensive data warehousng and analytics solution, from building data warehouse to generating actionable insigts,  I am designing this for my portfolio project to highlight industrys best practices in data engineering and analytics
#### Specifications 
- Data Sources : import  data from two  source systems(ERP,CRM) provided as CSV files
- Data Quality : Cleanse and resolve data quality issues  prior to analysis
- Integration  : Combine both sources into one user friendly data model for analytical  queries
- Scope        : Focus onto latest dataset, historization is not required  
## Objective 
Develop a modern data warehouse using SQL Server  to consolidate sales data, enabling  analytical reporting  and informed desicion making 
- customer behavior
- product performance
- sales trends

All of this will lead to business oriented decision making
